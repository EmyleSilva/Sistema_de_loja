void mensagem_final ( int  status_operacao)

Exibe uma mensagem no fim de cada operação informando o status final.

Informa ao usuario se a operação foi finalizada com sucesso (constante FINALIZADA) se a operação foi cancelada (constante CANCELADA) ou ainda se a operação falhou (constante FALHOU).

Parameters
status_operacaoConstante que indica o status final da operação.

void mostra_erro_e_encerra ( char  tipo[])

Exibe na tela uma mensagem de erro e encerra o programa.

Quando um erro interno ocorre (ERRO_MEMORIA, ERRO_ARQ_W, ERRO_ARQ_R), e o programa precisa ser encerrado, o usuário é informado sobre o tipo de erro que aconteceu,e logo após o programa é finalizado.
